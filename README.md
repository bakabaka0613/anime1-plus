# Anime1.me Plus

為 [anime1.me](https://anime1.me/) 打造的瀏覽器增強油猴腳本。

## 功能

- **自動封面圖（嚴謹匹配）**：解析標題（去集數/字幕組、辨識「第X季」「Ⅱ」「2nd Season」「劇場版」「OVA」等），到 [Bangumi](https://bgm.tv) 搜尋，用「名稱相似度 + 年份 + 季度」三維評分。**信心不足時不靜默對錯**，改列出候選讓你手動選，選擇會被記住。
- **列表頁封面縮圖**：首頁 `/` 動畫列表的名稱前自動顯示封面（lazy + 限流載入，避免打爆 Bangumi；信心不足者留虛線占位，進分類頁可手選）。
- **觀看記錄 + 續播**：在**分類頁就地內嵌播放器**看時，自動辨識你播的是第幾話並記錄進度（這是 anime1 實際的觀看方式）；單集頁 `/{postId}` 也支援。重看自動跳回上次位置；確實播到 ≥90% 才標記「看完」。分類頁每集標示「已看 ✓ / 進度條」，頂部橫幅顯示**「上次看到第幾話」**並可一鍵繼續看。
- **清除記錄**：油猴選單可「清除此動畫的觀看記錄」（修正先前殘留的錯誤標記）。
- **自動下一集**：看完倒數 5 秒自動播放下一集（可取消、可關閉）。
- **鍵盤快捷鍵**：`←/→` 快退/快轉 10 秒、`空白` 播放暫停、`N` 下一集、`F` 全螢幕、`+/-` 調速。
- **播放速度記憶**、**追番清單浮動面板**（右下 📺，列出未看完的動畫並可繼續看）。
- **資料匯出/匯入**：油猴選單提供 JSON 匯出/匯入，方便備份或搬機。

## 安裝

1. 瀏覽器安裝 [Tampermonkey](https://www.tampermonkey.net/) 或 Violentmonkey。
2. 安裝 `dist/anime1-plus.user.js`（拖入或新增腳本貼上內容）。
3. 打開任一動畫頁即生效。

## 開發

```bash
npm install
npm test          # 純函式單元測試（parse / match）
npm run build     # 產出 dist/anime1-plus.user.js
```

原始碼拆分於 `src/`（`parse` 標題解析、`match` 匹配評分、`bangumi` 搜尋、`store` 儲存、`progress` 觀看記錄、`ui` 介面、`cover` 封面協調、`main` 分派），由 esbuild 打包成單檔油猴腳本。anime1 DOM 選擇器集中在 `src/dom.js`，改版時優先檢查此處。

## 手動驗證清單

- [ ] 開含季度的動畫（如「為美好的世界獻上祝福！第三季」）→ 封面卡正確；同名不同季不會對錯。
- [ ] 名稱冷僻時 → 出現候選清單，手選後記住。
- [ ] 播放某集 30 秒 → 重整 → 自動續播跳回。
- [ ] 看完一集 → 標記已看完 + 倒數自動下一集。
- [ ] 分類頁 → 已看集數顯示 ✓、看一半顯示進度條。
- [ ] 油猴選單匯出 JSON → 匯入 → 記錄還原。

## 已知限制

- Bangumi 公開 API 有速率限制 → 結果以 `catId` 快取，避免重複請求。
- anime1 播放器若改為非原生 `<video>`，續播/記錄會自動略過（不報錯）。
- 列表頁名稱連結若用 `?cat=id` 而非 `/category/...` 路徑，縮圖的快取 key 會與分類頁不同（仍能顯示，只是不共用、會各搜一次）。
- 新番更新提醒列為後續迭代。
